<h2>Backend Setup</h2>
<ol>
  <li>Install dependencies using "npm install"</li>
  <li>Create .env with NODE_ENV=development, JWT_SECRET=thisissecret, MONGO_URI=mongodb://localhost:27017/dbName </li>
  <li>Run the backend with "npm run server"</li>
</ol>
